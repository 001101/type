IBM Type

The package of IBM’s new typeface, IBM Plex, and accompanying typography code. Find installation instructions on the Github repo.

Characters

To ensure that correct characters load in HTML, add the meta tag for UTC-8.

A common example of why this is necessary is when correcting a straight single quote (') for an actual apostraphe (’). 

<head>
  <meta charset="utf-8">
  ...
</head>

Styles

There are two prescriptive sets of the same type scale. Both variations start to differentiate at the size of Type D. You are able to toggle between type scales based on editorial and product use.

The type styles will apply font sizing, weight, line height, character spacing, and bottom margin for you. Please add a negative left margin when the line starts away from container padding. Further bottom margin specifications should be considered when pairing different parts of the scale together.

The type scale variations are fluid. This means that there are fixed sizes at the small, medium, and large breakpoints. However, the text sizes will increase and decrease between those screen width breakpoints for perfect sizing with the user's device.
