Line-length

We set up max-widths for text length based on the column width of IBM Grid, type scales and the limit of characters.

Note: According to The Elements of Typographc Style, 45 to 75 characters is widely regarded as a satisfactory length of line for a single-column page set in a serifed text face in a text size.

Alignment

Elements of good typography include consistency, hierarchy  and alignment. Flush left creates strong alignments for the eye to follow, aiding readability. It is our standard for text layout.

Flush left

Oops!

Flush right is reserved for languages that read right to left.

Oops!

Justified copy can fit into IBM typography, but we recommend not using this approach as it is difficult to do well and requires strong typographic expertise.

Bottom Margin

We set up bottom margin based on the type scales and weights. Use the increments of 2px (0.125rem) to define the bottom-margin for text pairings.

Fluid Type

To create a better experience along with our fluid grid, fluid type is the way to go. We set up our fix type scales for the breakpoints and have them being fluid in between. This helps us to maintain the quality of typography.

Why do we like fluid type?

It potentially creates better rags when you scale up the screen. It also provides a better line-length for reading.

Type Color

To maintain the quality and recognition of our work, we always consider how to use our color in design. Legibility and accessiblity are two keys. Keep our type color neutral when it shows in paragraphs. We never use pure black (#000) for text, instead, we use a range of grays.
